DBSteward PEAR Channel

Registering the channel: 
pear channel-discover pear.dbsteward.org
Listing available packages: 
pear remote-list -c dbsteward
Installing a package: 
pear install dbsteward/package_name
Installing a specific version/stability: 
pear install dbsteward/package_name-1.0.0
pear install dbsteward/package_name-beta
Receiving updates via a feed: 
http://pear.dbsteward.org/feed.xml

DBSteward - SQL database XML definition tool for DDL creation and differencing

Database definition tool for creation or upgrade of an existing database with a "SQL compiler" approach. Database upgrade DDL/DML/DCL is generated from source XML, without connecting to the target database. http://dbsteward.org/
